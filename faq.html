FAQ


Important Tips

1) Wait till the resources are ready in GENI before running the experiment in LabWiki:
Once the resources are reserved, its paramount to wait until the resources are ready which is indicated by the nodes turning green in the 'slice page' or status on each of the nodes turns 'READY'.

2) If the experiment in LabWiki is in a pending state for a long time:
If the experiment status in LabWiki is in 'pending' state for a long time, it is advised to retype the experiment name instance in the search bar in the 'Execute' page or reload the whole page.

General

1) How can I create a new assignment?
New experiments are created thru rpsecs which are usually already provided. The rspecs define the type of device being run(through images that are provided in the rspec), the number of nodes, define the interfaces and their IP addresses, etc.

2) Which GENI resource should I use?
The GENI resource to be used will be provided for each assignment and can be selected from 'Select Existing' drop down menu.

3) How do I know which rack(site) I need to use to run my experiment?
While it is assured that we could use any of the available racks (indicated with a green tick mark) can be used to run your experiments, please reach out to the instructor to make sure a rack available in the US is being run.

Troubleshooting

1) I don't see a graph in my experiment.
The graphs may not be generating for one of two reasons. First, if the X-axis and Y-axis paramaters are not defined correctly. These paramaters can be seen at the bottom of OEDL code. Secondly, if the servers running these processes are overloaded. The signs indicating such an issue is marked by a red cross next to the 'Received messages' caption in the execute window of LabWiki.

2) My experiment has switched from "Running" to "Aborted".
This can happen if there was a failure during execution of your experiment; For e.g., if one or more of the running applications failed or if connectivity to your testbed setup was lost. Check detailed logs for error messages.

3) My experiment has switched from "Pending" to "Aborted".
This error occurs if there are syntax errors in your OEDL script.
